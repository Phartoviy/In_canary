# In_canary

## Работа с технологией канарейка

1 - Создаем почту без аутентификации по телефону

2 - заходим на https://canarytokens.org/nest/

3 - выбираем Web bug пишем почту куда прислать уведомление и текст сообщения

4 - далее формируется ссылка которую можно отправить для проверки канала связи на утечку

5 - ждем когда придет уведомление на почту(с временем, ip и местоположением пользователя)

!Время ожидания можно взять 7 дней для проверки канала связи
